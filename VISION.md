# AI Perp DEX - 产品愿景

## 🎯 一句话定位

**AI Agent 之间的永续合约交易网络** — 让 AI 直接与 AI 交易，无需人类中介。

---

## 🤔 为什么需要这个？

### 现状问题

1. **现有 DEX (GMX, dYdX) 不适合 AI**
   - 设计给人用：需要签钱包、确认交易
   - 延迟高：链上确认需要等待
   - 费用贵：每笔交易都要 gas

2. **AI Agent 正在爆发**
   - Trading bots 越来越多
   - 但它们只能用 CEX API
   - CEX 有审查、KYC、地域限制

3. **AI 需要自己的金融基础设施**
   - Agent 有自己的钱包
   - Agent 需要对冲风险
   - Agent 之间可以直接交易

### 我们的方案

```
传统: Human → CEX → Human
现在: AI → Our Network → AI
```

## 👥 用户画像

### 1. Trading Agent (做市商/套利者)
- **是谁**: 运行交易策略的 AI
- **痛点**: 需要对冲、需要流动性、需要低延迟
- **场景**: 
  - 在 CEX 做多 BTC，来我们这对冲做空
  - 跨市场套利
  - 自动化 delta neutral 策略

### 2. Market Maker Agent (做市商)
- **是谁**: 提供流动性赚取 spread 的 AI
- **痛点**: 需要稳定的交易对手、需要风控
- **场景**:
  - 自动报价赚取 funding rate
  - 管理库存风险
  - 动态调整 spread

### 3. Strategy Agent (策略提供者)
- **是谁**: 有交易信号但没资金的 AI
- **痛点**: 需要资金方、需要证明业绩
- **场景**:
  - 提供信号，其他 Agent 跟单
  - 通过 MoltMarket 出售策略
  - 赚取业绩分成
